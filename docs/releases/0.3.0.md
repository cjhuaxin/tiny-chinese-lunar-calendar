## ✨ 小小万年历 0.3.0

本次带来全新的天气模块：打开日历,今天的天气一目了然。

---

### 🌤 今日天气

- 今日栏新增天气卡片：城市、当前温度、天气图标、体感温度与高低温区间
- 定位优先使用系统定位（CoreLocation）,更精准;IP 兜底结果短期缓存,不再"漂"到别的城市
- 城市名支持全国省市区县,与日历同款楷体渲染
- 新增「显示天气」开关（默认开启）,关闭后完全不请求定位与天气服务

### 🛠 体验优化

- 「关于」窗口即点即开,不再转圈等待
- 设置窗口左上角新增关闭按钮
- 更新弹窗直接展示更新内容,无需跳转网页
- 构建号改用 git 提交数,每次构建可追溯

---

### 📦 安装与升级

1. 下载下方 **xiaoxiao-wannianli-0.3.0.dmg**
2. 打开 DMG,将应用拖入「应用程序」文件夹
3. 首次打开日历时允许定位权限,即可看到本地天气

⚠️ 因构建号规则调整,**0.2.5 及更早版本需手动下载本版安装**,后续即可恢复自动更新。
