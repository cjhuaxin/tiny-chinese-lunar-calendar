## ✨ 小小万年历 0.2.3

轻巧的 macOS 菜单栏农历万年历，原生 Rust + Slint 渲染，常驻内存极低。

---

### 🎯 本版本亮点

**修复更新源版本滞后**

jsDelivr 的 `@main` 分支缓存不可靠，导致「检查更新」长期误报 0.2.1 为最新版。本版本将 `SUFeedURL` 改为 GitHub Releases 的 `releases/latest/download/appcast.xml`，始终指向最新发布的 appcast。

---

### 📦 安装与升级

**0.2.2 及更早版本请手动安装本版 DMG**（旧版更新源无法自我修复）。

1. 下载下方 **xiaoxiao-wannianli-0.2.3.dmg**
2. 打开 DMG，将应用拖入「应用程序」文件夹
3. 点「检查更新」，应显示「已是最新版本 v0.2.3」
