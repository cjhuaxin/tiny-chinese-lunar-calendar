## ✨ 小小万年历 0.2.1

轻巧的 macOS 菜单栏农历万年历，原生 Rust + Slint 渲染，常驻内存极低。

---

### 🎯 本版本亮点

**修复「检查更新」失败**

0.2.0 的更新源指向 `raw.githubusercontent.com`，在国内部分网络环境下会因 SSL 错误或连接超时导致 Sparkle 无法获取升级信息。本版本将 appcast 源切换至 [jsDelivr CDN](https://www.jsdelivr.com/)，国内访问更稳定，「检查更新」可正常工作。

---

### 📦 安装与升级

#### 从 0.2.0 升级

0.2.0 无法通过应用内更新自我修复，请手动下载并安装本版 DMG 覆盖旧版本。安装后，后续版本将正常通过应用内更新推送。

1. 下载下方 **xiaoxiao-wannianli-0.2.1.dmg**
2. 打开 DMG，将应用拖入「应用程序」文件夹
3. 右键菜单栏图标 → **检查更新**，确认显示「已是最新版本」

#### 新用户

1. 下载下方 **xiaoxiao-wannianli-0.2.1.dmg**
2. 打开 DMG，将应用拖入「应用程序」文件夹
3. 从菜单栏点击图标即可使用
