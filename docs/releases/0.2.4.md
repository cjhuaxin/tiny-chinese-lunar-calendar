## ✨ 小小万年历 0.2.4

轻巧的 macOS 菜单栏农历万年历，原生 Rust + Slint 渲染，常驻内存极低。

---

### 🎯 本版本亮点

**完善检查更新体验**

- 更新状态弹窗复用系统 About 面板样式，与「关于」窗口一致
- 修复关闭弹窗后程序意外退出的问题
- 设置、关于、检查更新窗口在点击日历后正确置顶显示

**网络与菜单优化**

- 检查更新前探测连接超时（10s），直连失败时自动尝试本地代理 `127.0.0.1:7890`
- 托盘菜单新增线条风格图标，调整菜单顺序（设置 → 检查更新 → 关于 → 退出）

---

### 📦 安装与升级

**0.2.3 及更早版本** 可通过应用内「检查更新」自动升级，或手动下载 DMG 安装。

1. 下载下方 **xiaoxiao-wannianli-0.2.4.dmg**
2. 打开 DMG，将应用拖入「应用程序」文件夹
3. 右键菜单栏图标 → **检查更新**，确认弹窗样式正常
