## ✨ 小小万年历 0.2.2

轻巧的 macOS 菜单栏农历万年历，原生 Rust + Slint 渲染，常驻内存极低。

---

### 🎯 本版本亮点

**优化「检查更新」弹窗**

重新设计更新状态提示：图标与文字垂直居中，文案精简为「已是最新版本」+ 版本号，去掉冗余的「版本历史记录」按钮。

**修复更新源缓存延迟**

发布流程新增 jsDelivr 缓存刷新，确保 appcast 更新后客户端能立即获取最新版本信息，不再误报旧版本为最新。

---

### 📦 安装与升级

从 **0.2.1 及更早版本** 升级，请下载并安装本版 DMG。0.2.2 起可通过应用内更新自动推送后续版本。

1. 下载下方 **xiaoxiao-wannianli-0.2.2.dmg**
2. 打开 DMG，将应用拖入「应用程序」文件夹
3. 右键菜单栏图标 → **检查更新**，确认弹窗样式与版本号正确
