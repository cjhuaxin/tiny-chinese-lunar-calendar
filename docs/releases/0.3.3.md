## ✨ 小小万年历 0.3.3

更新体验与定位授权专项修复。

---

### 🔄 更新体验修复与增强

- 修复手动"检查更新"时窗口卡在"正在检查更新"的问题
- 后台发现新版本后,打开日历时会弹出更新对话框,由你决定是否升级,不再全程静默
- 同一版本每次运行只提醒一次,选择"稍后"不会被反复打扰

### 📍 定位授权不再重复弹窗

- 应用改用固定开发者身份签名,定位授权在版本升级后继续有效
- **升级到本版本后,系统会最后询问一次定位授权**,此后的版本更新不再重复询问

### 🎨 细节修正

- 今天格子的天气图标与顶部今日天气保持一致(均为实时天气)

---

### 📦 安装与升级

- **0.3.x 用户**:点击菜单栏图标 → 检查更新,即可自动升级
- **0.2.5 及更早版本**:请手动下载下方 **xiaoxiao-wannianli-0.3.3.dmg** 安装,之后即可恢复自动更新
