## ✨ 小小万年历 0.2.5

轻巧的 macOS 菜单栏农历万年历，原生 Rust + Slint 渲染，常驻内存极低。

---

### 🎯 本版本亮点

**检查更新不再「没反应」**

- 点击「检查更新」后立即弹出「正在检查更新 / 请稍候…」提示（复用 About 面板样式）
- 检查完成后同一窗口切换为「已是最新版本」或 Sparkle 更新窗口
- 30 秒网络超时后明确提示「网络连接失败，请检查网络或代理」
- 启动时在 Sparkle 初始化前预配置网络；直连 GitHub 失败时自动尝试本地代理 `127.0.0.1:7890`
- 延迟 90 秒再执行后台自动检查，避免与手动检查冲突

---

### 📦 安装与升级

**0.2.4 及更早版本** 可通过应用内「检查更新」自动升级，或手动下载 DMG 安装。

1. 下载下方 **xiaoxiao-wannianli-0.2.5.dmg**
2. 打开 DMG，将应用拖入「应用程序」文件夹
3. 右键菜单栏图标 → **检查更新**，确认立即出现「正在检查更新」提示
