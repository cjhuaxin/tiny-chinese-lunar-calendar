## ✨ 小小万年历 0.2.0

轻巧的 macOS 菜单栏农历万年历，原生 Rust + Slint 渲染，常驻内存极低。

---

### 🎯 本版本亮点

**应用内自动更新**

集成 [Sparkle](https://sparkle-project.org/) 更新框架，从此无需每次手动下载 DMG。应用启动后会在后台静默检查新版本，发现更新后自动下载，并通过原生对话框提示安装——一键完成升级。

**托盘菜单新增「检查更新」**

右键点击菜单栏图标，选择 **检查更新**，可随时手动触发更新检查，查看当前版本与最新版本的对比。

**安全签名的更新包**

所有更新包均经 EdDSA 签名验证，应用内置公钥校验，确保你收到的更新来自官方渠道，防止中间人篡改。

---

### 📦 安装与升级

#### 新用户

1. 下载下方 **xiaoxiao-wannianli-0.2.0.dmg**
2. 打开 DMG，将应用拖入「应用程序」文件夹
3. 从菜单栏点击图标即可使用

#### 从 0.1.x 升级

0.1.x 版本尚未内置自动更新，请手动下载并安装本版 DMG 覆盖旧版本。安装完成后，后续版本将通过应用内更新自动推送，无需再手动下载。

> 首次运行若被 Gatekeeper 拦截，请在「系统设置 → 隐私与安全性」中允许打开。

### 🛠 技术变更

- 新增 `src/updater.rs` 模块，通过 `sparklers` 封装 Sparkle 框架
- 应用启动时自动检查更新，并启用后台静默下载
- `Info.plist` 配置 `SUFeedURL` 与 `SUPublicEDKey`，指向 GitHub 托管的 appcast 源
- 构建流程新增 Sparkle 框架打包、更新 zip 生成与 appcast 签名脚本
- 版本号与 `CFBundleVersion` 统一由 `scripts/lib/version.sh` 从 semver 自动映射
